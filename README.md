# Programacion-Funcional
## Trabajo Practico:Programacion Funcional
### Contenido del Trabajo Practico
Este trabajo práctico se enfoca en aplicar la API de Streams de Java para procesar y analizar distintas colecciones de datos, usando como modelo las clases Alumno, Producto, Libroy Empleado.

El objetivo es usar un pipeline funcional para resolver varios requerimientos. Vamos a implementar filtros para seleccionar datos específicos, como solo los alumnos aprobados, los productos que superan cierto precio, los libros con más de 300 páginas o los empleados con salarios altos.

Una parte clave es la generación de estadísticas y reportes . Para esto, calcularemos promedios generales (de notas, precios, páginas y salarios) y también encontraremos valores extremos, como el libro más caro o los 3 mejores promedios.

También vamos a practicar mucho la agrupación de datos usando Collectors, organizando los elementos por curso, categoría, autor o departamento. Dentro de esas agrupaciones, realizaremos más cálculos, como sumar el stock total por categoría, contar la cantidad de libros por autor o sumar los salarios de cada departamento.

Finalmente, las operaciones de TP incluyen para transformar y ordenar los datos, como pasar nombres a mayúsculas, limitar resultados para obtener (por ejemplo) los 2 empleados más jóvenes, y generar una cadena única con los datos de los productos.
